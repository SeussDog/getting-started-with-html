The Web
• Collection of HTML docs
• Hyperlinks connect all the HTML docs

HTML
• Hyper Text Markup Language
o Text
o Markup
o Ref to other docs (ie pics and vids)
o Links to other pages

• Markup
o Tags
• < Name > Contents < /Name >
• < b > = Bold Tag
• < em > = Emphasis or Italic Tag
o Elements
• The contents that sits between tags
o Attributes
• < Tag Attr = "value" > contents < /tag >
• Anchors are for making links
• Anchor = < a >
• < a href = "www.reddit.com"> derp < /a>
o Images
• < img> = images
• < img src = "url" alt = "text"> <--- This is for broken link messages
o Line Break
• < br>
• < p> = paragraph
o Inline vs Block
• < br> = inline
 Just lines and text
• < p> = block
 Makes an invisible box

• < span> = inline
• < div> = block
